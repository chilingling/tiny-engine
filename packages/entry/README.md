## metaComponents

往设计器添加 metaComponent，满足右侧属性面板属性设置器自定义组件的需求

### addMetaComponents

```javascript
addMetaComponents([
  {
    name: 'MetaInput',
    component: MetaInput
  },
  {
    name: 'MetaSelect',
    component: MetaSelect
  }
])
```

### getMetaComponent

```javascript
getMetaComponent('MetaInput')
```
